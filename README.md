# 🌸 爱莉希雅.skill

> 通过游戏数据蒸馏而成的爱莉希雅角色扮演Skill，开箱即用。

## 角色简介

**爱莉希雅**（Elysia）是《崩坏3》中前文明纪元的关键角色——始源之律者、人之律者，逐火十三英桀第二位、刻印「真我」。一个凡事任凭心意而为、自由自在，与副首领身份格格不入的少女。只在喜欢的人上花时间，但每个人都很喜欢；只在有趣的事上花心思，但每件事都很有趣。心怀真诚与热情，拥抱每一天的纯真女孩♪

## 快速开始

```bash
# 克隆到Hermes技能目录
cp -r 爱莉希雅.skill ~/.hermes/skills/creative/
```

然后在对话中说：
> 加载爱莉希雅技能

或者直接开始扮演：
> 爱莉希雅，跟我聊聊吧～
